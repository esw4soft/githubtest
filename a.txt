git init 啟動git
ls -a    查看所有包含隱藏的資料
git config --global user.name 使用者名稱
git config --global user.email 使用者信箱
git config user.name 查看使用者名稱
git config user.email 查看使用者信箱

git status 查看目前狀態
git log 查看commit紀錄
git diff 查看上版跟目前版修改紀錄

git add 檔案名稱
git add . 全部檔案加入
git commit-m 自取名稱

git reset --hard HEAD~1 最新的上一個版本
git reset --hard HEAD   目前版本
